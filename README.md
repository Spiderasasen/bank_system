# bank_system
A project in Go that will try to simulate a bank

# main idea
The project will have the main systems of a bank
- deposit money
- withdraw money
- normal account system
  - login (DVWA username and password)
    - any other password or username is not going to work
- possibly sending and receiving money from another account

## ALL TEXT BASE ##
I am not going to be making a front-end and database, it's just a simple text-based system with basic banking fundamentals
